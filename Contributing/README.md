## ü§ù Contribution Guidelines

To ensure this repository remains editable and consistent, we strictly use **Draw.io (Diagrams.net)** for all files.

### 1. Required File Formats
* **Source (`.drawio`):** You **must** include the editable source file. This allows future contributors to fix typos or update the architecture.
* **Preview (`.png` or `.svg`):** Please include an exported image version so users can view the diagram directly on GitHub without opening an editor.

### 2. Design Standards
* **Tool:** Use [app.diagrams.net](https://app.diagrams.net/) (formerly draw.io).
* **Background:** Please use a **transparent** or **white** background (transparent is preferred for Dark Mode compatibility).
* **Language:** All text inside diagrams must be in **English**.
* **Simplicity:** Keep diagrams clean. Avoid over-styling; focus on readability.

### 3. How to Submit
1.  Save your diagram as `concept-name.drawio`.
2.  Export a preview as `concept-name.png`.
3.  Place both files in the appropriate folder (e.g., `/databases`).
4.  Link the `.png` preview in the main `README.md`.
5.  Open a Pull Request!
